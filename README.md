
# Desti

## Overview
Desti is a full-stack car pool application with a Node.js/Express backend and a React/Vite frontend.

## Project Structure
- `backend/`: Node.js + Express REST API
- `frontend/`: React + Vite web client

## Getting Started

### Backend
See `backend/README.md` for setup instructions.

### Frontend
See `frontend/README.md` for setup instructions.

## Features
- User management
- Car pool creation and joining
- RESTful API
- Modern UI with React
